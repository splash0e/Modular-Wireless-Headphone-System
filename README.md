This work proposes a Modular Wireless Headphone System that transforms legacy Bluetooth headphone electronics into a reusable external audio controller. 
The proposed architecture combines the Bluetooth motherboard of a Panasonic RP-BTD5 headset with Sennheiser HD 202 wired headphones, a higher-capacity 
lithium-ion battery, and a modern USB Type-C charging interface. The modular design enhances repairability, facilitates future hardware upgrades, 
promotes sustainable reuse of electronic components, and extends battery operating time while maintaining reliable wireless audio functionality.
